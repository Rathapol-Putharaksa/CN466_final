## Line and LIFF

# Line

ทำหน้าที่รับ message 

เมื่อได้message check จะตอบกลับสถานะปัจจุบันว่าอยู่ในสถานะ stay หรือ move


เมื่อได้message time จะตอบกลับเวลาทั้งหมดที่ใช้ในสถานะ stay และ move
